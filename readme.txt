OBJETIVO DEL SISTEMA DE INFORMACIÓN
El objetivo del presente sistema informático es lograr gestionar de manera eficiente los recursos y servicios entregados a nuestros clientes. Permitiendo que sus usuarios: clientes, profesionales y trabajadores administrativos puedan registrar y obtener información importante para poder funcionar de manera organizada y oportuna.
